# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Multi-language UI support (English, Turkish, German, Finnish)
- Prompt history with scrollable view
- Clear prompt history functionality
- Stop generation feature with abort controller
- Language selector component
- Internationalization (i18n) system with translation hooks

### Changed
- Improved system prompt with NEXUS-10 architecture
- Enhanced prompt refinement with context-aware tech stack detection
- Better error handling for Ollama connection
- Updated UI animations and visual effects
- Improved prompt parsing and extraction logic

### Removed
- Prompt quality indicator component (UI simplification)
- Prompt improvements component (UI simplification)
- Mode selector component (replaced with dropdown)
- Unused prompt analysis services
- Legacy documentation files

### Fixed
- Fixed metadata injection into AI prompts
- Fixed language detection and output matching
- Fixed stop button z-index and visibility issues
- Removed unused imports and state from store

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Context-aware prompt refinement
- Ollama integration
- Project folder analysis
- Tech stack detection
- Dark/light theme support
- Electron desktop application
- React + TypeScript + Vite setup
